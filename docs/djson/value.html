    Module djson.value

    Core data structures and API for djson. This module defines JValue, JObject, and JArray types.

    Classes

    NameDescription
    JSONException Exception thrown on JSON parsing or traversal errors.
    JSONPartialException Exception thrown specifically when the JSON parser requires more data from an ongoing stream to complete the current operation.
    JSONSyntaxException Exception thrown specifically for syntax errors where the JSON format is natively invalid or corrupted.

    NameDescription
    JArray Represents a JSON Array (ordered list of values).
    JObject Represents a JSON Object (ordered mapping of keys to values).
    JValue Represents a single JSON value. Uses a union to store different types efficiently and supports lazy evaluation.
    SafeResult Wrapper for a result that might not exist. Used by the .safe!T() methods to avoid throwing exceptions.
