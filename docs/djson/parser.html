    Module djson.parser

    Internal and public parsing logic for djson. This module implements lazy parsing, skipping, and full eager parsing.

    Functions

    NameDescription
    parseJSON(data) Main entry point to parse a JSON string. Returns a JValue that will be parsed lazily as fields are accessed.
    parseJSONComplete(data) Fully parse JSON string eagerly (no lazy evaluation). Returns a completely parsed JValue in a single pass — faster than parseJSON + parseAll().
    skipValue(s) Skips the current JSON value and updates s to point after it. It works iteratively by maintaining depth.
    stripJSONWhitespace(s) Helper function to strip leading JSON-standard whitespace. Whitespace includes space, tab, newline, and carriage return.
